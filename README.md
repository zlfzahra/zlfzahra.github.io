# Zalfa Afifah Zahra Academic Portfolio

Personal academic portfolio template for GitHub Pages.

## How to use

1. Upload all files to `zlfzahra.github.io`
2. Put your profile photo in `assets/profile.jpg`
3. Put your CV in `assets/cv.pdf`
4. Replace `#` links in HTML with your real LinkedIn, Google Scholar, ResearchGate, and ORCID URLs.
5. Commit changes.
6. Open `https://zlfzahra.github.io`

## Pages

- `index.html`
- `about.html`
- `projects.html`
- `publications.html`
- `contact.html`
